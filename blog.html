Important FAQ

What is the Document Object Model (DOM)?

The DOM is a tree-like structure that represents a webpage and allows JavaScript to change its content, structure, and styles, allowing for dynamic interactions.

How do you select an element from the DOM?

To select an element from the DOM, there are various JavaScript methods:

  • Select an element by its ID:
    document.getElementById("elementId");
  • Select elements by their class name:
    document.getElementsByClassName("className");
  • Select the first element by querySelector:
    document.querySelector(".className");
  • Select all elements by querySelectorAll:
    document.querySelectorAll(".className");
What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM means handling events on a parent element instead of individual child elements. It's useful for efficiency, especially with dynamically generated content.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using JavaScript DOM methods:

  • Set or change an attribute:
    element.setAttribute("attribute", "value");
  • Get an attribute's value:
    element.getAttribute("attribute");
  • Remove an attribute:
    element.removeAttribute("attribute");
  • Change an element's style (inline CSS):
    element.style.color = "blue";
  • Add a class to an element:
    element.classList.add("newClass");
  • Remove a class from an element:
    element.classList.remove("oldClass");
  • Toggle a class on or off:
    element.classList.toggle("activeClass");
